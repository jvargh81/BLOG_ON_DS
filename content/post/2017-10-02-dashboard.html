To use Dashboard, we also have to use one more package called ‘flexdashboard’:
  • Using Dashboards we can plot different graphs together.

Lets plot some graphs for using it in the Dashboard.

Adding Librarys

library(flexdashboard)
library(Lahman)
library(dplyr)
library(ggplot2)

Extracting the data I want.

result<-Batting%>%
  filter(playerID=='ruthba01')%>%
  select(yearID,SO,HR)
  • Lets plot first graph based on Strike out and Homeruns

_ visualize the data using scatter plot._

ggplot()+
  geom_point(data=result,aes(x=SO,y=HR))+
  xlab("Strike outs")+
  ylab("Homeruns")

  • Lets plot second graph based on Homeruns _ visualize the data using Line plot._
ggplot()+
  geom_histogram(data=result,aes(x=HR),color='white',bins=15)+
  xlab("Homeruns")

  • Lets plot third graph based on Homeruns and yearID. _ visualize the data using point and line plot._
ggplot()+
  geom_point(data=result,aes(x=yearID,y=HR))+
  geom_line(data=result,aes(x=yearID,y=HR))+
  xlab("Year")+
  ylab("Homeruns")
